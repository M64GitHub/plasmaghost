# TSDEMO

## BUILD

```bash
cd yourbuilddir
git clone https://github.com/M64GitHub/libtsprites.git
git clone https://github.com/M64GitHub/tsdemo.git
cd tsdemo
make
```

## RUN

```bash
./tsdemo
```

## CONTROL

- Use cursor keys / vi keys to move up / right / down / left
- ESC / q to quit
- CTRL-C to quit
