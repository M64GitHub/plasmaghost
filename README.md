# PLASMAGHOST

## BUILD

```bash
cd yourbuilddir
git clone https://github.com/M64GitHub/libtsprites.git
git clone https://github.com/M64GitHub/plasmaghost.git
cd plasmaghost
make
```

## RUN

```bash
./plasmaghost
```

## CONTROL

- Use cursor keys / vi keys to move up / right / down / left
- ESC / q to quit
- CTRL-C to quit
